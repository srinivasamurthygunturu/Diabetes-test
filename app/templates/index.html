Summary

After performing extensive EDA on the data set a significant number of interesting insights were discovered out of which class imbalance problem is the most important charaterisic of this dataset. We used SMOTE tecchnique to overcome this problem. To achieve our goal we had build 5 different models namely Logistic Regression, KNN, Decision Trees, Random Forests,SVM. Random Forest Classifier out performed the rest of the models and had the highest precission value. From this exercise we can conclude that it is possible to predict diabetes using non invasie methods aswell. However, the performce of the model can be imporoved by augmenting more meaningful data.
